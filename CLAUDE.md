## What is PlatFormer?

A Kubernetes-native platform engineering tool that abstracts cloud infrastructure
complexity. Developers declare what kind of app they have (serverless, microservices,
monolith) and PlatFormer provisions all the required cloud infrastructure automatically.

**Stack:** Go (backend, controllers, CLI) · React + TypeScript (UI) · AWS (v1 cloud target)
**Model:** Open source core → paid enterprise tier
**Status:** Phase 1 MVP — Week 1 of 12

## Architecture Rules (NEVER Violate These)

1. **Controllers never import cloud SDKs directly.**
   Any AWS import found in `internal/controller/` is an architecture violation.
   All cloud calls go through the `CloudProvider` interface.

2. **The provider interface is the contract.**
   `internal/provider/provider.go` must be defined before any AWS code is written.

3. **Generic types only in the provider package.**
   No AWS-specific types (`types.Runtime`, `aws.String`, etc.) allowed outside
   of `internal/aws/`.

4. **MockProvider for all controller/CLI testing.**
   Never use real AWS credentials in unit tests. Wire MockProvider instead.

5. **Finalizers on every CRD.**
   Every controller must register a finalizer to prevent orphaned AWS resources
   on deletion.

6. **CRD status is the source of truth.**
   The CLI and UI read CRD `.status` fields. Never return data via stdout only.

## The CloudProvider Interface (Reference)

```go
// internal/provider/provider.go
type CloudProvider interface {
    CreateFunction(ctx context.Context, spec FunctionSpec) (*FunctionResult, error)
    UpdateFunction(ctx context.Context, spec FunctionSpec) (*FunctionResult, error)
    DeleteFunction(ctx context.Context, name string) error
    GetFunction(ctx context.Context, name string) (*FunctionResult, error)

    CreateAPIEndpoint(ctx context.Context, spec APISpec) (*APIResult, error)
    UpdateAPIEndpoint(ctx context.Context, spec APISpec) (*APIResult, error)
    DeleteAPIEndpoint(ctx context.Context, id string) error
    GetAPIEndpoint(ctx context.Context, id string) (*APIResult, error)

    CreateExecutionRole(ctx context.Context, spec RoleSpec) (*RoleResult, error)
    DeleteExecutionRole(ctx context.Context, name string) error
    GetExecutionRole(ctx context.Context, name string) (*RoleResult, error)

    CreateDatabase(ctx context.Context, spec DatabaseSpec) (*DatabaseResult, error)
    DeleteDatabase(ctx context.Context, name string) error
    GetDatabase(ctx context.Context, name string) (*DatabaseResult, error)

    CreateLogGroup(ctx context.Context, name string) error
    DeleteLogGroup(ctx context.Context, name string) error
}
```
